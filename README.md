# 📊 Customer Churn Prediction System

An end-to-end Machine Learning project that predicts whether a telecom customer is likely to churn based on Personal Details, service, and billing information.

## 🚀 Features

* End-to-end ML pipeline
* Data preprocessing and feature engineering
* Multiple model comparison
* Hyperparameter tuning using GridSearchCV
* Threshold optimization
* Interactive Streamlit dashboard
* Dynamic business recommendations
* Customer risk factor analysis
* Prediction report download

## 📈 Model Performance

* Algorithm: Logistic Regression
* Test Accuracy: 80.70%
* Cross Validation Accuracy: 80.44%
* Decision Threshold: 0.55

## 💡 Business Insights

Customers are more likely to churn when they:

* Use Fiber Optic internet
* Have Month-to-Month contracts
* Use Electronic Check payment methods
* Have shorter tenure periods
* Have higher monthly charges
